# Installation

    $ npm install

    $ mongod

    $npm run start

# API Users

## Overview
It's CRUDS api for managed all users.

### [POST] Create user
Allows the creation of a single user.

|                            |                  |
|----------------------------|------------------|
| Requires authentication ?  | No               |
| Who can use it ?           | Owner and users  |
| Response formats           | application/json |

* HTTP request : POST → user/create

#### Parameters :
